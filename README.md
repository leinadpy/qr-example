## Description

Nest with QR code generator

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

Try to do a http request GET to 'http://localhost:3000/qr-code?data=URL' where URL will be the url for the QR-code.
